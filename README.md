# Overview
The repo contains my solution code to CSW186's 6 projects from SQL query to a Java-based relational database RookieDB and MongoDB's query(MQL). 

## Project 0 
- Environment setup for Java-based database RookieDB.

## Project 1 
- SQL query exercise on SQLite.

## Project 2
- Implementation of the B+ Tree as the index in the RookieDB.

## Project 3
- Implementation of several join algorithms in RookieDB (Block Nested Loop Join, Grace Hash Join, Sort Merge Join).

- Implementation of a naive query optimizer in RookieDB based on Dynamic Programming. 

## Project 4
- Implementation of concurrency control in RookieDB based on 2-phase lock.

## Project 5
- Implementation of ARIES recovery system on RookieDB

## Project 6
- MQL query exercise on MongoDB.
